Project description

This project was done in order to help my friend, the founder of permafforest.fr, to plant and restore forests using the Miyawaki method. This method of ecological engineering today known as the "Miyawaki method" aims to restore native forests from seeds of native trees.

One of its conditions for success is "Plantations randomly distributed in space in the way plants are distributed in a clearing or at the edge of the natural forest, not in rows or staggered (meeting on this point with the Prosilva methods in Europe)." But by using randomness we can end up with multiple trees from the same layer at the same spot, which would make them compete for resources (sunlight, space...etc...) and therefore not growing the best way. This program tries to avoid that edge case as much as possible in order to have the healthiest and fastest growing forest.
