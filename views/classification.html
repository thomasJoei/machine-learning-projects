Project description

This project aims to automatically classify customer reviews in categories. The original dataset contains 1 000 000 reviews (100 000 reviews for each category). The model used is a Naive Bayes classifier, which has been trained on 80% of the dataset.

Text classification can have many real life applications and use cases :

  • Automate some simple email responses of an online customer service (re-send train tickets, redirect user to correct FAQ section, ...etc...)
  • Detect sensible content on social media platforms
  • Estimate users feedback from comments (how was the content received by the users on a Facebook post? Did they hated it? Liked it? Neutral feeling?...etc...)
  • For more uses cases check this out : https://towardsdatascience.com/text-classification-applications-and-use-cases-beab4bfe2e62
